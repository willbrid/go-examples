### Les opérateurs de base en Go

- `+, -, *, /, %`

Ces opérateurs permettent d'effectuer des opérations arithmétiques sur des valeurs numériques. L'opérateur `+` peut également être utilisé pour la concaténation de chaînes de caractères.

- `==, !=, <, <=, >, >=`

Ces opérateurs comparent deux valeurs.

- `||, &&, !`

Ce sont les opérateurs logiques, qui sont appliqués aux valeurs booléennes et renvoient une valeur booléenne.

- `=, :=`

Ce sont les opérateurs d'affectation. L'opérateur d'affectation standard `=` sert à initialiser une constante ou une variable, ou à modifier la valeur d'une variable déjà définie. L'opérateur abrégé `:=` permet de définir une variable et de lui affecter une valeur.

- `-=, +=, ++, --`

Ces opérateurs incrémentent et décrémentent des valeurs numériques.

- `&, |, ^, &^, <<, >>`

Ce sont les opérateurs bit à bit, applicables aux valeurs entières. Ces opérateurs sont rarement utilisés dans le développement courant. <br>
Par exemple l'opérateur `|` est utilisé pour configurer les fonctionnalités de logging de Go.
